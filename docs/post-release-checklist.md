# Post-release checklist

Run within 1–24h after production deploy.

## Performance
- Check PageSpeed Insights (mobile) for the homepage
- Verify real-world load on a mid-tier phone

## Forms
- Submit test form(s)
- Confirm deliverability (inbox + spam)
- Confirm any webhooks/CRM entries are created

## Links
- Run link checker (or spot-check key paths)
- Verify redirects (http→https, www→non-www)

## Analytics (if enabled)
- Verify pageview events
- Verify outbound click / form submit events (if configured)
