# QA (Role: QA)

This folder contains lightweight QA artifacts: checklists, runbooks, and report templates.

## Quick start (when repo is available)
1. Provide one of:
   - Repo URL + branch/PR
   - Or a deployed URL (staging/prod)
2. Then we will run:
   - Smoke: Home → Projects → Project Detail → Contact
   - Lighthouse mobile (Perf ≥ 90) + check LCP/CLS/INP
   - Reduced-motion checks
   - A11y keyboard/focus/headings/alt/aria/skip-link
   - Cross-browser matrix
   - SEO (meta/OG/canonical/sitemap/robots/404)

## Files
- `mvp-smoke-checklist.md` — primary checklist
- `test-report-template.md` — fill per build
- `bug-report-template.md` — file per issue
